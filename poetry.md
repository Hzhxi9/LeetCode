## 雨帘

```
不知道你在远方是不是也在望着这场雨。
我用很短的时间，就记录了一些事物的过程。然后对着天空，把自己读成雨帘之外模糊的身影。
所有的往事都流动起来，汇成一个苍白的声音，这种感觉由来已久，可是没有哪一次能穿越雨帘。
我知道，这场雨过后，今天便成亘古了。而我希冀的那种声音，在每场过去的雨中，回荡不己。
```

```
风，揭开了窗前一部分黑暗，更深的黑暗又飘落下来。
我站在窗口，经受着这阵风，它在到来前，是否经过一棵开花的樱桃树？一只风筝？小河边鹅卵石上的月光？它们连同一根草绳样路，被风吹引到今夜。
今夜，弯曲的灯光更加弯曲。而想着的一个人，身披尘土，从我的身体里走出。
```

```
翻开所有情节，都是鲜艳的玫瑰。许多次，试探着想挤进某个段落，成为其中的标点符号。
的小你一直在精心编织着春天，让阳光和草地，挤走时间里的阴雨。我准备好了雨伞，却没有打开，铺满树叶的道路尽头，是别人居住的小屋。
的小清贫的爱情居无定所，就像我的那些文字，在无奈中学会了沉默，而你的花香、鸟鸣和果实，被装订进你的青春，成为永久的珍藏。

```

夜雨在落

墙边的花

```

```
